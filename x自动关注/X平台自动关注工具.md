# X平台自动关注工具

## 功能介绍

这是一个用于自动化处理X平台(原Twitter)关注操作的Python工具。该工具可以：

- 从CSV文件中批量读取用户名
- 自动构建并打开X平台搜索页面
- 实现模拟人类行为的自动点击操作
- 跳过第一个用户，固定处理接下来的80个用户
- 使用8-14秒的随机延迟，使操作更符合人类行为模式

## 工具特点

- **智能搜索**：自动构建搜索URL，精确查找目标用户
- **模拟人类行为**：鼠标移动路径自然，包含中间点和微小调整
- **随机化处理**：使用随机延迟和移动，避免被识别为自动化脚本
- **固定点击位置**：使用预设的X=1120, Y=270坐标进行精准点击
- **详细日志**：全程记录操作过程，便于监控和调试

## 环境要求

- Python 3.6+
- 依赖库：`pyautogui`

安装依赖：
```bash
pip install pyautogui
```

## 使用方法

### 基本用法

最简单的使用方式是直接运行脚本，使用默认配置：

```bash
python x刷粉.py
```

这将：
- 使用默认CSV文件：`e:\MedicalGPT-main\TwExporter-mengmenghenkeai-344-followers.csv`
- 跳过第一个用户，处理接下来的80个用户
- 启用自动点击功能
- 使用8-14秒的随机等待时间

需要配合插件使用：
TwExporter获取关注列表，插件地址：https://github.com/mengmenghenkeai/TwExporter
然后根据对应的关注列表去关注就行，等对方回关。
